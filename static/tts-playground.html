TTS Playground

Standalone sandbox — not wired into the Bullpen app. Served from /static.

1. Web Speech API free built-in

The browser's native speechSynthesis. Voices come from the OS, so the list differs per machine. Zero download, zero cost.

2. Kokoro.js (WebGPU) free 100% local

An 82M-param neural model running fully client-side via Transformers.js. First load downloads ~300 MB (cached afterward). Needs a WebGPU browser (Chrome/Edge, recent Safari); falls back to slower WASM otherwise.
